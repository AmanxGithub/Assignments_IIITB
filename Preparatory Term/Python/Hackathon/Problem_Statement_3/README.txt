Run the .py file in Python 2.7 environment.
The user will be asked his/her lms account username and password.
Next the user will be asked a sender Email address username and password
Since google SMTP server has been used, the sender might have to grant 3rd party access to his/her email account.
Next the user will be asked a recipient email address.

The recipient may check his/her inbox folder as well as the spam folder.
